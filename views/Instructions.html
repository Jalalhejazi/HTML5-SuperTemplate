You can download using git, npm or bower

git installation
git clone https://github.com/Jalalhejazi/HTML5-SuperTemplate.git
npm installation
npm install html5-supertemplate
bower installation
bower install HTML5-SuperTemplate
Head Setup
<link rel="stylesheet" href="SuperTemplate.css" media="all" />
Body Setup
<script src="http://code.jquery.com/jquery-latest.min.js"></script>
<script src="SuperTemplate.js"></script>

Browsers

HTML SuperTemplate Tested and working in IE 8+, Safari, Chrome, Firefox, Opera, Safari IOS, Browser and Chrome Android.

Notes

Don't forget to use an HTML5 Doctype <!DOCTYPE html>
